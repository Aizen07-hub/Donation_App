# **App Name**: PlateShare

## Core Features:

- Listing Creation: Easy listing creation with details on food type, quantity, and pickup time.
- Browse Listings: Display nearby restaurants with available surplus food.
- Real-time Alerts: Notifications for new listings from preferred restaurants or food categories.
- Smart Scheduling: Suggest optimal pickup times considering factors like restaurant's closing time using an AI tool.

## Style Guidelines:

- Primary color: Earthy Green (#4CAF50) to convey freshness and sustainability.
- Secondary color: Light Beige (#F5F5DC) for a clean background.
- Accent color: Warm Gold (#FFC107) for highlighting important actions.
- Clean and modern typography that is easy to read
- Use food-related icons
- Subtle animations on loading.
